Manga Scraper

16 November 2014

I'm quite a fan of japanese manga, you can read them online for free but sometimes its best to have a local copy on your ipad or laptop.

Rather than download each issue individually I've create a tool that will scrap all the issues for a manga from manga panda. You can find the source code at my github

About the Project

The project is written in python and uses beautifulsoup.
